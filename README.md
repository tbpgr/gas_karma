gas_karma

## 目的
感謝管理ツール

## 機能一覧
|command|description|
|---|---|
|/karma thx <name1> <name2>|name1, name2 に感謝を送る。感謝を送る相手は何名でも可|
|/karma list|現在の徳ランキングを表示する|
